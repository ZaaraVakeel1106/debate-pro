# Debate Tournament Manager

A web-based system to manage debate tournaments efficiently.  
Built using Flask and SQLite.

---

## Features

- Team registration
- Automated match generation
- Judge scoring system
- Round progression
- Leaderboard with rankings
- Bracket visualization
- BYE handling for odd teams

---

## How It Works

1. Add teams from Admin panel  
2. Start tournament (Round 1 generated automatically)  
3. Judges enter scores  
4. System progresses rounds automatically  
5. Leaderboard updates based on performance  
6. Bracket shows match progression  

---

## Tech Stack

- Python (Flask)
- SQLite
- HTML, CSS (Bootstrap)
